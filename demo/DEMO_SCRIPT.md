# ADIA 3-Minute Demo Script

## 0:00 - 0:30 | Problem Setup
- Introduce ADIA as an Autonomous Decision Intelligence Agent.
- Enter a sample scenario:
  - "Should a startup launch a food delivery app in Hyderabad?"

## 0:30 - 1:10 | Agentic Architecture
- Explain the 4-agent flow:
  - Research Agent -> Analysis Agent -> Reasoning Agent -> Report Agent
- Mention Nova usage:
  - Max 3 Nova calls per request (cost-safe).

## 1:10 - 1:50 | Multimodal Input
- Upload one document (PDF/text/image) as evidence.
- Click **Analyze**.
- Highlight that ADIA uses PyPDF2 + FAISS locally for evidence retrieval.

## 1:50 - 2:35 | Output Walkthrough
- Show viability score chart.
- Explain top risks and opportunities.
- Show recommendation and summary.
- Open **Reasoning Timeline** to show transparent AI decision steps.

## 2:35 - 3:00 | Hackathon Value
- Reinforce practical use cases:
  - startup strategy, market expansion, pricing decisions, risk review.
- Reiterate free-tier friendly architecture:
  - local execution + Bedrock Nova only.
